üåê Deploy to Netlify (Without Git)

  1. Run locally first: npm install && npm run build
  2. Drag & drop the `dist` folder to Netlify.com
  3. Set environment variables: SESSION_SECRET (required) + Firebase vars (optional)
  4. Deploy! Your app is live
